unlockActiveWallet

suspend fun unlockActiveWallet(password: String): Boolean

Unlocks the active wallet for use with sensitive SDK operations. This function or unlockWallet must be called before calling sensitive operations like payInvoice.

Return

True if the wallet was unlocked successfully, false otherwise.

Parameters

password

The password for the wallet.

Throws

LightsparkException

If the wallet ID is not set yet.
